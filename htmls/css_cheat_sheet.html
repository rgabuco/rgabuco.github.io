CSS Cheat Sheet

Selectors

Selects the HTML element to style

selector { property: value; }

Class Selector

.class { property: value; }

ID Selector

#id { property: value; }

Universal Selector

* { property: value; }

Descendant Selector

parent child { property: value; }

Adjacent Sibling Selector

element1 + element2 { property: value; }

Attribute Selector

[attribute=value] { property: value; }

Pseudo-class Selector

selector:pseudo-class { property: value; }

Grouping Selector

selector1, selector2 { property: value; }

List Styles

List elements can be styled using various properties to change their appearance and layout.

list-style-type, list-style-image, list-style-position, list-style

Text Properties

Control text alignment, decoration, transformation, and spacing

text-align, text-decoration, text-transform, letter-spacing

Background Properties

Set background color, image, size, position, and repeat

background-color, background-image, background-size, background-position

Border Properties

Define border width, style, and color

border-width, border-style, border-color

Font Properties

Set font family, size, weight, style, and more

font-family, font-size, font-weight, font-style

Margin and Padding

Control spacing outside and inside an element

margin, padding

Width and Height

Set the width and height of an element

width, height

Box Model

Box model is essential for layout design. It consists of content, padding, border, and margin.

content-box, padding-box, border-box

Display Property

Change how elements are displayed in the layout

display

Positioning

Control the position of an element

position, top, right, bottom, left

Flexbox

Layout mode for arranging elements in a container

display: flex, flex-direction, justify-content, align-items

Grid

Layout mode for creating grid-based designs

display: grid, grid-template-columns, grid-template-rows

Transforms

Apply transformations like rotation, scaling, and skewing

transform, rotate, scale, skew

Transitions

Add smooth animations to CSS properties

transition, transition-property, transition-duration
