HTML Cheat Sheet

Document Structure

HTML documents are structured using elements, which are marked up with tags.

<!DOCTYPE html> <html> <head> <title> <body>

Meta Tags

Meta tags provide metadata about the HTML document.

<meta charset="UTF-8"> <meta name="description" content="description">

HTML Layouts

HTML layouts are created using different elements such as:

<header>, <nav>, <section>, <article>, <aside>, <footer>

Heading Tags

HTML has six levels of headings, from <h1> to <h6>, where <h1> is the highest level.

Paragraphs

Paragraphs are defined with the <p> tag.

Links

Links are created with the <a> (anchor) tag.

<a href="url">link text</a>

Images

Images are inserted using the <img> tag.

<img src="image.jpg" alt="description">

Lists

Lists can be ordered (<ol>) or unordered (<ul>).

Unordered List (Bullet Points):

<ul>
  <li>Apples</li>
  <li>Bananas</li>
</ul>

Ordered List (Numbered):

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

Tables

Tables are created with the <table>, <tr>, and <td> tags.

Table with headers:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
</table>

Span

The <span> tag is used to group inline-elements in a document.

<span>

Comments

Comments in HTML are written inside <!-- comment --> tags and are not displayed in the browser.

<!-- This is a comment -->

Forms

Forms are used to collect user input and are created with the <form> tag.

Text input:

<input type="text" name="username">

Password input:

<input type="password" name="password">

Email input:

<input type="email" name="email">

Checkbox:

<input type="checkbox" name="vehicle" value="Car"> I have a car

Radio buttons:

<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female

Textarea:

<textarea name="message"></textarea>

Select dropdown:

<select name="cars"> <option value="volvo">Volvo </option> <option value="saab">Saab </option> </select>

Submit button:

<input type="submit" value="Submit">

Attributes

HTML elements can have attributes that provide additional information about them.

Example Attribute Usage:

Adding an ID to an element:

<div id="myDiv"></div>

Adding a class to an element:

<p class="highlight"> Highlighted text</p>

Setting the width and height of an image:

<img src="example.jpg" alt="Example Image" width="100" height="100">

Divisions

A web page's content is usually divided into blocks, specified by the <div> tag.

Inline vs. Block Elements

Inline elements do not start on a new line and only take up as much width as necessary, whereas block elements start on a new line and take up the full width available.

Inline Elements:

<span> <img> <a>

Block Elements:

<p> <div> <h1>

Linking CSS Stylesheet

To apply styles to your HTML document, you can link an external CSS stylesheet using the <link> tag.

<link rel="stylesheet" href="styles.css">

Adding JavaScript

To add interactivity and behavior to your HTML document, you can include JavaScript code either inline or by linking an external JavaScript file.

Inline JavaScript:

<script>alert('Hello, World!');</script>

External JavaScript:

<script src="script.js"></script>

Adding Videos and Audios

HTML5 introduced the <video> and <audio> tags to embed media directly into web pages.

Embedding a Video:

<video src="video.mp4" controls>Video</video>

Embedding Audio:

<audio src="audio.mp3" controls>Audio</audio>
