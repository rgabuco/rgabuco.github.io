JavaScript Cheat Sheet

Variables

Declaring and assigning variables:

let x = 5;
const PI = 3.14;
var y = "Hello";

Conditional Statements

Using if, else if, and else:

let age = 18;
if (age >= 18) {
alert("You are an adult.");
} else {
alert("You are a minor.");
}

Loops

Iterating with for and while loops:

for (let i = 0; i < 5; i++) {
console.log(i);
}
let i = 0;
while (i < 5) {
console.log(i);
i++;
}

Objects

Creating and accessing object properties:

let person = {
name: "John",
age: 30
};
console.log(person.name);

Functions

Defining and calling functions:

function greet(name) {
return "Hello, " + name + "!";
}
alert(greet("John"));

String Methods

Using built-in string methods:

let message = "Hello, world!";
console.log(message.toUpperCase());
console.log(message.length);

Event Handling

Attaching event listeners to elements:

document.getElementById("myButton").addEventListener("click", function() {
alert("Button clicked!");
});

Arrow Functions

Using concise syntax for functions:

const add = (a, b) => a + b;
console.log(add(3, 5)); // Output: 8

Template Literals

Using backticks to create multi-line strings:

const name = "John";
console.log(`Hello, ${name}!`); // Output: Hello, John!

Arrays

Creating and accessing arrays:

let colors = ["red", "green", "blue"];
console.log(colors[0]);
colors.push("yellow");

Asynchronous JavaScript

Working with asynchronous code using promises:

fetch ("https://api.com/data")
.then(response => response.json())
.then(data => console.log(data));

DOM Manipulation

Accessing and modifying HTML elements:

let element = document.getElementById("myElement");
element.textContent = "Hello, world!";

Selecting Elements

Use document.getElementById() to select an element by its ID.

Use document.querySelector() to select the first element that matches a CSS selector.

Use document.querySelectorAll() to select all elements that match a CSS selector.

Traversing the DOM

Use parentNode to access the parent node of an element.

Use childNodes to access a collection of child nodes.

Use previousSibling and nextSibling to access adjacent nodes.

Use firstChild and lastChild to access the first and last child nodes.

Manipulating Elements

Use createElement() to create a new element.

Use appendChild() to append a child element to a parent element.

Use removeChild() to remove a child element from its parent.

Use setAttribute() to set attributes on an element.

Use innerHTML to get or set the HTML content of an element.
